# Профиль пользователя

> Ваши личные настройки. Здесь можно изменить пароль, обновить контактные данные и настроить предпочтения уведомлений.

## Личные данные

В этом блоке вы можете изменить информацию, которую видят коллеги и (в некоторых случаях) клиенты:
- **Имя и Фамилия**: Отображаются в чатах и отчетах.
- **Email**: Используется для входа и уведомлений.
- **Аватар**: Загрузите фото для персонализации профиля.
- **Telegram**: Привяжите свой аккаунт Telegram для получения уведомлений и входа через Telegram Login.

## Безопасность

### Смена пароля
Для изменения пароля введите текущий пароль, затем новый пароль и его подтверждение.
*Совет: Используйте надежные пароли, содержащие буквы разного регистра, цифры и спецсимволы.*

### Сессии
Вы можете посмотреть список устройств, с которых выполнен вход в ваш аккаунт, и при необходимости завершить подозрительные сессии.

## Уведомления

Настройте, какие события вы хотите отслеживать и где:
- **Каналы**: Email, Telegram, Браузер (Push).
- **События**:
  - Новое сообщение в моем чате.
  - Назначение меня ответственным.
  - Нарушение SLA (для менеджеров).
  - Новый отзыв.

## Интерфейс

- **Тема**: Светлая / Темная / Системная.
- **Язык**: Выберите язык интерфейса (Русский / English).
