## Технические требования

- **Формат**: SVG (векторный) + PNG (1024x1024, прозрачный фон)
- **Варианты**: светлый фон, тёмный фон, монохром
- **Favicon**: 32x32, 180x180 (Apple Touch)
- **Цветовая палитра**:
  - Primary: `#6366f1` (Indigo)
  - Secondary: `#8b5cf6` (Purple)
  - Accent: `#a855f7` (Pink)
  - Glow: `rgba(99, 102, 241, 0.4)`

---

## Примечания для генерации

- Используй **Midjourney v6** или **DALL-E 3** для лучших результатов
- Добавь `--style raw` в Midjourney для чистого дизайна
- После генерации — доработай в Figma/Illustrator для идеальных векторов
- Проверь читаемость на размерах 16px, 32px, 64px, 256px
