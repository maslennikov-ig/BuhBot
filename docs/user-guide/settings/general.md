# Общие настройки

> Основные параметры системы: рабочее время, календарь праздников и настройки уведомлений.

_Доступно только администраторам._

## Рабочий график

Настройка режима работы вашей службы поддержки. Эти данные критически важны для корректного расчета SLA.

### Часы работы

Укажите время начала и окончания рабочего дня для каждого дня недели.

- _Пример: Пн-Пт с 09:00 до 18:00._
- Если в какой-то день вы не работаете, снимите галочку "Рабочий день".

### Праздники и выходные

Добавьте даты государственных праздников или корпоративных выходных. В эти дни таймеры SLA будут остановлены.

### Часовой пояс

Установите часовой пояс, в котором работает ваша команда (например, Europe/Moscow). Все отчеты и графики будут отображаться по этому времени.

## Настройки SLA (по умолчанию)

Здесь задаются глобальные стандарты обслуживания:

- **Целевое время первого ответа**: (например, 15 минут).
- **Целевое время решения**: (например, 8 часов).

Эти настройки применяются ко всем новым чатам и запросам, если не переопределены индивидуальными правилами для VIP-клиентов (если такой функционал активен).

## Системные уведомления

Настройка глобальных каналов оповещения для администраторов.

- Email для ежедневных отчетов.
- Telegram-чат для критических алертов системы.
