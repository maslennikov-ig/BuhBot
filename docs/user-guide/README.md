# Руководство пользователя BuhBot

> Платформа автоматизации коммуникаций для бухгалтерских фирм

## Введение

BuhBot — это система управления клиентскими коммуникациями через Telegram, разработанная специально для бухгалтерских компаний. Платформа позволяет централизованно обрабатывать обращения клиентов, контролировать качество обслуживания и автоматизировать рутинные задачи.

## Быстрый старт

### Вход в систему

1. Откройте веб-интерфейс BuhBot по адресу, предоставленному администратором
2. Авторизуйтесь через Telegram, нажав кнопку «Войти через Telegram»
3. Подтвердите вход в Telegram-боте

## Роли пользователей

### Оператор

- Отвечает на сообщения клиентов
- Использует шаблоны быстрых ответов
- Создаёт и обрабатывает запросы

### Менеджер

- Все возможности оператора
- Просмотр аналитики и отчётов
- Контроль SLA и нарушений
- Управление командой операторов

### Администратор

- Все возможности менеджера
- Настройка системы
- Управление пользователями
- Конфигурация шаблонов и FAQ

## Навигация по интерфейсу

### Боковое меню

Основная навигация расположена в левой части экрана. Разделы сгруппированы по функциональности:

- **Главные**: Дашборд, Чаты
- **Операционные**: Запросы, Алерты, Нарушения
- **Мониторинг**: SLA, Аналитика, Отзывы
- **Система**: Настройки

### Контекстная справка

На каждой странице есть кнопка **?** рядом с заголовком. Нажмите её, чтобы увидеть краткое описание текущего раздела.

### Уведомления

Системные уведомления отображаются в правом верхнем углу. Критические алерты требуют немедленного внимания.

## Горячие клавиши

| Клавиша        | Действие                     |
| -------------- | ---------------------------- |
| `Ctrl + K`     | Быстрый поиск                |
| `Esc`          | Закрыть модальное окно       |
| `Enter`        | Отправить сообщение (в чате) |
| `Ctrl + Enter` | Отправить с переносом строки |
