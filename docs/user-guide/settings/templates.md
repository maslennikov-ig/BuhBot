# Шаблоны ответов

> Управление шаблонами быстрых ответов. Создавайте заготовки для частых ситуаций, используйте переменные для персонализации.

## Обзор

Шаблоны позволяют операторам отвечать быстрее и качественнее, используя стандартизированные формулировки компании.

## Создание шаблона

1. Нажмите "Создать шаблон".
2. **Название**: Краткое имя для поиска (например, "Приветствие", "Запрос документов").
3. **Текст**: Содержание сообщения.
4. **Категория**: Группировка для удобства (Общие, Отчетность, Технические).
5. **Доступность**: Личный (только для меня) или Общий (для всей команды).

## Переменные

Для персонализации сообщений используйте переменные, которые автоматически заменятся на реальные данные при отправке:

- `{{clientName}}` — Имя клиента (например, "Иван").
- `{{operatorName}}` — Ваше имя (например, "Мария").
- `{{companyName}}` — Название вашей компании.

*Пример шаблона:*
`Здравствуйте, {{clientName}}! Меня зовут {{operatorName}}. Чем могу вам помочь?`

## Использование

Созданные шаблоны становятся доступны операторам в окне чата. Их можно вызывать через меню шаблонов или горячими клавишами (если настроено).

> **Совет**: Создавайте шаблоны для самых частых сценариев: приветствие, прощание, запрос реквизитов, уведомление о сроках подготовки отчета.
