# Аналитика

> Детальные отчеты о работе сервиса. Изучайте тренды нагрузки, продуктивность сотрудников и динамику клиентской удовлетворенности.

## Обзор

Раздел "Аналитика" предоставляет инструменты для глубокого изучения данных о работе службы поддержки. Это помогает принимать обоснованные управленческие решения, планировать нагрузку и находить точки роста.

## Доступные отчеты

### 1. Нагрузка (Volume)
Показывает количество входящих обращений (чатов, запросов).
- **По дням/неделям**: Помогает выявить пиковые дни.
- **По времени суток**: Помогает планировать смены операторов.

### 2. Продуктивность операторов (Performance)
Сравнение эффективности сотрудников.
- Количество закрытых запросов.
- Среднее время ответа.
- Оценки от клиентов (CSAT).

### 3. Распределение запросов (Distribution)
Статистика по тематикам и типам проблем.
- Какие вопросы задают чаще всего (на основе тегов).
- Распределение по приоритетам.

### 4. Тренды времени ответа
Динамика скорости реакции поддержки.
- Улучшается или ухудшается сервис со временем?

## Фильтры

Все отчеты можно гибко настраивать:
- **Период**: Выберите стандартный (Сегодня, Неделя, Месяц) или произвольный диапазон дат.
- **Оператор**: Данные по всей команде или конкретному сотруднику.
- **Тип запроса**: Анализ только определенных категорий задач.

## Работа с графиками

- Наводите курсор на точки графика, чтобы увидеть точные значения.
- Отключайте/включайте отдельные линии легенды кликом, чтобы сравнить показатели.

## Экспорт данных

Если встроенных графиков недостаточно, вы можете выгрузить "сырые" данные или готовые отчеты:
- **CSV**: Для анализа в Excel или Google Таблицах.
- **PDF**: Готовый отчет для руководства.
