# Настройки системы

> Обзор раздела настроек и конфигурации платформы BuhBot.

## Обзор

Раздел настроек позволяет адаптировать систему под бизнес-процессы вашей компании. Доступ к определенным подразделам зависит от роли пользователя.

## Структура настроек

### Личные настройки (Доступно всем)
- [Профиль](./profile.md): Изменение пароля, уведомлений, личных данных.

### Настройки компании (Только Администраторы)
- [Общие настройки](./general.md): График работы, SLA, часовой пояс.
- [Пользователи](./users.md): Управление сотрудниками и правами доступа.

### Настройки автоматизации (Администраторы и Менеджеры)
- [Опросы](./survey.md): Конфигурация NPS и CSAT опросов.
- [Шаблоны](./templates.md): Управление быстрыми ответами.
- [База знаний (FAQ)](./faq.md): Настройка автоответчика бота.

## Роли и доступ
